# Weather-App-project
A responsive weather application that provides real-time weather updates based on the user's searched city. The app displays key information such as temperature, humidity and weather conditions using a clean and user-friendly interface. Built using HTML, CSS, and JavaScript, and integrated with a weather API for live data.
